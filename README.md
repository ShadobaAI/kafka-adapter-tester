# 1С: Адаптер Kafka — Тестер

![Платформа](https://img.shields.io/badge/1С-8.3.21+-blue)
![EDT](https://img.shields.io/badge/EDT-2025.2+-blue)

Тестовое расширение конфигурации с примерами использования API и сценариями отладки интеграции для [1С: Адаптер Kafka](https://github.com/ShadobaAI/kafka-adapter).

## Назначение

Репозиторий содержит расширение конфигурации, которое демонстрирует типовые сценарии работы с адаптером:

- **примеры произвольных обработчиков** — реализации функций сериализации (1С → Kafka) и десериализации (Kafka → 1С);
- **примеры вызовов публичного API** — прямая отправка и чтение сообщений через `кфкИнтеграция`;

> Расширение предназначено только для разработки. В промышленную эксплуатацию не поставляется.
